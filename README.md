## Setup
- Add `nuxt-graphql-cms` dependency using yarn or npm to your project
- Add `nuxt-graphql-cms` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-graphql-cms',

    // With options
    ['nuxt-graphql-cms', { /* module options */ }],
 ]
}
```
